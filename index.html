What the Master DB holds

The armf-master D1 database collates data from two sources:
① Cage Tool DB — all cage, frame, movement, shipment, factory and audit data (read via CAGE_DB binding)
② Remote DB — the analyses table pulled in via the Cloudflare API

The cage tool and cage traveller apps now read mesh weights and cage refresh data from here instead of syncing the remote DB themselves. Hit Sync Master above to pull the latest analyses from the remote DB.
